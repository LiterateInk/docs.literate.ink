## What is a restaurant?

A restaurant is a place where you can find food.
There's multiple restaurants in each Crous, you can retrieve restaurants using the `restaurants(identifier)` method where `identifier` is the Crous identifier - that you can find using `feeds()`.

## Listing restaurants

<Tabs items={['JS/TS', 'Kotlin', 'Rust']}>
```ts tab="JS/TS"
import * as crous from "crowous";

const identifier = "limoges";
const restaurants = await crous.restaurants(identifier);

for (const restaurant of restaurants) {
  console.log(restaurant.title);
  console.log("=>", restaurant.address);
  console.log();
}
```
 
```ts tab="Kotlin"
import ink.literate.crowous.api.restaurants

suspend fun main() {
  val identifier = "limoges"
  val restaurants = restaurants(identifier)

  for (restaurant in restaurants) {
    println(restaurant.title)
    println("=> ${restaurant.address}")
    println()
  }
}
```
